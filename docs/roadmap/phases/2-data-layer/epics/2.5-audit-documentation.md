## Technical Notes

### PRD Connection Pooling Specs

**PostgreSQL:**

- MaxOpenConns: 25
- MaxIdleConns: 5
- ConnMaxIdleTime: 5 minutes
- ConnMaxLifetime: 1 hour

**SQLite/go-libsql:**

- MaxOpenConns: 1 (single writer)
- MaxIdleConns: 1
- WAL mode enabled
